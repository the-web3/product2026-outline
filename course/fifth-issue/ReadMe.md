# The Web3 社区第五期 Web3 产品投研班开启报名--熊市学习拉开差距，牛市来了决定位置


## 一. 课程概述
本课程面向希望系统进入 Web3 行业、提升产品认知与投研能力的人群，构建从区块链底层 → 产品架构 → 投研分析 → 真实产品设计 → 代币模型与合规的完整能力体系。

不是知识拼盘，而是一次从认知到实战能力的系统升级。

The Web3 社区是全球领先的 Web3 教育与孵化平台，已培养：
- 300+ 优秀开发工程师

选择我们，你获得的不只是课程，而是进入行业核心产研圈层的通道。

一线实战讲师阵容： 汇聚公链核心开发者、交易所高管、DeFi 先驱及专业投研分析师，将行业真实经验与系统方法论深度结合。
真实产业资源对接：链接主流交易所、公链生态与核心技术社区，让优秀学员直达项目机会、投资资源与合作通道。

紧跟行业主线赛道：围绕 Layer1、 Layer2、Layer、LSD、RWA、AI+Web3、AA 钱包、Web3 钱包、MPC资管、发射平台、事件预测平与模块化区块链，聚焦正在发生的结构性机会。
行业人脉与项目实战机会： 与开发者、产品经理和投资人深度协作，在真实项目拆解与设计中实现能力落地与资源积累。
课程持续进化机制：The Web3 社区始终与市场同步迭代课程体系，确保学习内容永远站在行业前沿。

## 三.本课程核心特色
体系化认知构建：从区块链底层逻辑到高阶产品投研，建立完整 Web3 认知框架，让不同基础的学员都能找到清晰成长路径。
产品级深度拆解:  基于 DappLink 等真实案例，系统拆解钱包、DEX、LSD、跨链桥、AA、MPC、CloudHSM、AI Agent、CCIP、发射平台和事件预测平台等核心产品体系。
实战驱动能力输出：通过投研报告、PRD、产品架构设计与原型图产出训练，真正形成「能分析 + 能设计」的实战能力。
专业数据工具实操：熟练使用 DeBank、Dune、Nansen、Glassnode、Messari 等工具，掌握链上数据获取、可视化分析与风险监控方法。
赛道级深度研究能力：覆盖公链、Layer2、DeFi、RWA、GameFi、AI+Web3、发射平台和事件预测平台等主流方向，建立跨赛道理解与趋势判断能力。
